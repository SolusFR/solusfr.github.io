## 🚀 Comment apparaître ici ?

Vous utilisez **Solus Wiki** pour votre serveur ? Nous serions ravis de vous ajouter à cette liste pour vous donner de la visibilité !

### Critères d'éligibilité :
1.  **Utilisation active** : Avoir un lien vers le wiki dans vos salons `#règlement` ou `#infos`.
2.  **Respect de la charte** : Votre serveur doit respecter les principes de base du règlement de Solus Wiki.
3.  **Partenariat** : Un échange de visibilité est apprécié mais non obligatoire.

### Formulaire de demande (Via Ticket) :
Pour être ajouté, ouvrez un ticket sur le Discord de **SolusFR** avec les infos suivantes ou envoyez un mp à angmarroisorcie sur Discord :
* Nom du serveur.
* Type (Minecraft, Communautaire, RP, etc.).
* Lien d'invitation permanent.
* Quelles pages du Wiki vous utilisez principalement.
